Keyboard shortcuts

Enter - sends the text to the text field.

Alt + 1-4 - changes the writing mode of the text.

Alt + Backspace - delete the last block of text.

Alt + clicking on a block highlights it for changes

Alt + R - Allows you to edit the text while leaving the same type of block with which it is written

Alt + E - Allows you to edit text by changing the block type.

Alt + S - loading saved text in local storage.

Alt + d - deleting saved text local storage.
